# Style Rwanda - E-commerce Platform

A complete e-commerce platform for Rwandan fashion store.

## 🌟 Features

- 🛍️ Product catalog with categories (Men, Women, Shoes, Accessories)
- 🛒 Shopping cart functionality
- 💳 Checkout with MTN MoMo & Airtel Money
- 📧 Email notifications (welcome, order confirmation, status updates)
- 🔐 User registration & login
- 🔑 Password reset via email
- 👑 Admin dashboard
- 📦 Order management (verify, ship, deliver, delete)
- 🖼️ Product image upload (camera, gallery, drag & drop)
- 📱 Mobile responsive
- 📨 Newsletter subscription
- 💬 Contact form with email notifications

## 🛠️ Technologies

- **Frontend:** HTML5, CSS3, Vanilla JavaScript
- **Backend:** PHP 8+
- **Database:** MySQL
- **Email:** PHPMailer (Gmail SMTP)
- **Server:** XAMPP / Docker

## 📋 Requirements

- PHP 8.0+
- MySQL 5.7+
- XAMPP / WAMP / MAMP
- Composer (for PHPMailer)
